# Password Strength Checker

## Description
This Python project checks the strength of a password based on security rules.

## Features
- Checks password length
- Detects uppercase and lowercase letters
- Detects numbers
- Detects special characters

## How to Run
python checker.py

## Example
Password@123

## Output
Strong Password
